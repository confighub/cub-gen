# Agentic GitOps Docs

This folder is organized for fast review and sharing.

Hard naming rule:

1. Use `Agentic GitOps` only when an active GitOps reconciler loop (`WET -> LIVE`)
   is present (Flux/Argo or equivalent).
2. Without that inner loop, call it `governed config automation`.
